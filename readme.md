# SoulSync: Emotional Support AI Assistant

## Overview
This project implements an AI-powered emotional support assistant using the InternVL2_5-1B vision-language model. The system allows users to upload images of themselves along with text describing their emotional state or concerns, and receives supportive responses generated by the AI.

## Features
- Web-based interface for easy interaction
- Image and text input capabilities
- Text and speech output capabilities
- AI-generated emotional support responses
- Real-time processing

## Technical Stack
- **Backend**: Flask (Python)
- **AI Model**: InternVL2_5-1B vision-language model
- **TTS Model**: ElevenLabs turbo-v2 
- **Frontend**: Streamlit
- **Image Processing**: PIL, Torch

## Installation

### Prerequisites
- Python 3.8+
- CUDA-compatible GPU (recommended)
- 16GB+ RAM

### Setup
1. Clone the repository:
   ```
   git clone https://github.com/your-username/v2r-soulsync.git
   cd v2r-soulsync
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the backend application:
   ```
   python main.py
   ```
4. Run the frontend application:
   ```
   streamlit run frontend.py
   ```
5. Access the web interface at `http://localhost:8051`

## Usage
1. Open the web interface in your browser
2. click on 'speak'  button to interact
3. Or Enter your concerns or emotional state in the text field
4. Instantly receive an AI-generated supportive response aligned to human-like values
